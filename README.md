# Pong Game

A quick vibe coded demo of the classic Pong arcade game built with React and TypeScript.

## How to Play

**Left Player:**
- W - Move paddle up
- S - Move paddle down

**Right Player:**
- Arrow Up - Move paddle up
- Arrow Down - Move paddle down

**Controls:**
- Space - Pause/Resume game

First player to 10 points wins!

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
